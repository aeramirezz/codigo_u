# Código U 🔴⚪

> *"Descifrando la historia del más grande."*

Proyecto de análisis de datos de **Universitario de Deportes**. Usamos Python y datos reales para entender al equipo más allá de lo que se ve en la cancha.

Todo lo que publicamos en [@CodigoU1924](https://x.com/CodigoU1924) nace de aquí.

## ¿Qué analizamos?
- Rendimiento por partido (antes y después de cada juego)
- Estadísticas de jugadores
- Histórico del club
- Fichajes y mercado de pases

## Herramientas
- Python
- API-Football
- pandas, matplotlib
